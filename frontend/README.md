# Frontend (React + Vite)

Este frontend usa React con archivos `.jsx` y Vite como bundler.

## Scripts
- `npm run dev` — inicia el servidor de desarrollo.
- `npm run build` — crea el build de producción.
- `npm run preview` — sirve el build creado.

## Estructura
- `index.html` — documento base.
- `src/index.jsx` — punto de entrada React.
- `src/App.jsx` — layout y rutas.
- `src/pages/Home.jsx` — página de inicio.
- `src/pages/Menu.jsx` — ejemplo de página de menú.

## Cómo iniciar
```powershell
cd "C:\Users\Usuario\Desktop\Restaurante Clase\frontend"
npm install
npm run dev
```

Luego abre la URL que muestre Vite (por defecto http://localhost:5173).
